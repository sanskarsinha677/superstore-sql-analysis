# Superstore Sales SQL Analysis

## Project Overview
This project analyzes Superstore sales data using SQL to extract meaningful business insights related to sales performance, profitability, and product categories.

## Tools & Technologies
- SQL (MySQL)
- Microsoft Excel (data cleaning & preparation)
- Git & GitHub

## Key Analysis Performed
- Overall business performance (sales & profit)
- Category-wise sales and profit analysis
- Top and bottom products by profitability
- Discount impact on sales and profit
- Quantity vs profit evaluation

## Files in this Repository
- `superstore_analysis.sql` – Contains all SQL queries used for analysis
- `README.md` – Project documentation

## Outcome
This project demonstrates practical SQL skills, analytical thinking, and the ability to convert raw business data into actionable insights.
